# ToolloopFront

## Comandos

- `npm install` para instalar las dependencias del proyecto.
- `npm run start` para iniciar el servidor de desarrollo.
- `npm run build` para construir el proyecto para producción.



##### Crear un servicio:
```
ng g s core/services/api/auth.api.service
```

##### Crear un componente de tipo página:
```
ng g c pages/login-page
```

##### Crear un componente de tipo compartido:
```
ng g c shared/components/header
```

#### Crear un componente de tipo layout:
```
ng g c shared/layouts/main-layout
```


### Dependencias Librerías
#### Para manejar cookies
```
ng add tailwindcss
npm install ngx-cookie-service --save
npm i @fortawesome/angular-fontawesome --save
npm install @fortawesome/free-solid-svg-icons
npm install @fortawesome/free-regular-svg-icons
```
